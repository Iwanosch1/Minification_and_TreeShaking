# Minification_and_TreeShaking

In diesem Branch (master) sind alle Dateien und in denen zu Minification bzw TreeShaking die jeweilg dazugehörigen Dateien aufgespaltet.

Link zu dem Vortrag:
https://docs.google.com/presentation/d/1QRFWBgugIL5XgZT8WQxpBVcsQaRjwNMnrL0YMngX8gM/edit?usp=sharing


Des Weiteren ist immer node.js (https://nodejs.org/en/) notwendig welches optimaler weise in der lokalen Ordnerstruktur installiert wird. Es ist auch möglich global zu installieren aber nicht gänig. Für TreeShaking ist zusätzlich webpack von nöten, im Archiv node_modules.tar stecken die lokalen Dateien, welche im gleichen Ordner liegen müssten, welche allerdings auch mit der Installtion von node.js bzw webpack lokal angelegt werden.

Genaue weitere Informationen zu dem jeweiligen sind in den Banchen in der README zu finden.
